 📡 Wireless Security Analysis using ESP8266

🔍 Overview

This project focuses on wireless network security testing using ESP8266 in a controlled lab environment. It explores WiFi vulnerabilities and evaluates mitigation techniques such as WPA2 and 802.11w.

 🛠️ Tools Used

* ESP8266 (NodeMCU)
* Kali Linux
* Wireshark (basic analysis)
* Nmap

 ⚙️ What I Did

* Configured ESP8266 for wireless testing
* Simulated deauthentication scenarios in a lab setup
* Studied network behavior and security weaknesses
* Researched mitigation techniques like Management Frame Protection

 🔐 Note

All testing was conducted in a **controlled and authorized environment** for educational purposes only.
